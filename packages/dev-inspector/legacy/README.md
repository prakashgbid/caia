# Legacy dev-inspector source (v0.1.0)

This directory holds the previous `@chiefaia/dev-inspector` implementation (formerly
`@pokerzeno/dev-inspector` v0.1.0): a hover-overlay element inspector with stable component
fiber IDs and click-to-copy.

It was superseded on 2026-04-28 by the tabbed inspector panel that lives at the parent
`src/`. See `../CHANGELOG.md` for the full rationale.

These files are excluded from `tsconfig.build.json` and from the package's exported surface.
They exist as a record only.
