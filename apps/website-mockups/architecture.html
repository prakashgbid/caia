Walkthrough

A single request, end to end.

"Add OAuth login with Apple + Google" — what actually happens between the operator pressing Submit and the green checkmark in CI.

  1. 01

    Operator submits intent

    Dashboard /submit posts free-form prose to the events bus, with project + domain context. Mentor's pre-spawn hook prepends precedent.

    → dashboard · mentor-retrieval · librarian

  2. 02

    PO decomposes

    decomposer-recursive splits into Initiative/Epic/Module/Story/Task. feature-registry classifies new-vs-enhance (sqlite-vec, zero Claude tokens).

    → decomposer-recursive · feature-registry · classifier

  3. 03

    Phase-1 agents fill the ticket

    BA, EA, DBA, BFF, UI, Security, Testing, Release each populate a Zod section of ticket-template. Validator rejects any malformed handoff.

    → ticket-template · agent-contract-registry · validator

  4. 04

    Chain runner dispatches

    phases.yaml is realised as a state machine. A LaunchAgent wakes the runner; it acquires a lock, emits a heartbeat, and spawns the coding worker.

    → chain-runner · chain-scaffolder

  5. 05

    Coding agent writes the diff

    Runs in an isolated worktree. local-llm-router picks the cheapest model that can satisfy the task (often a local 7B); escalates to Claude when complexity demands.

    → claude-spawner · local-llm-router · llm-cache · prompt-optimizer

  6. 06

    Four-layer safety stack

    Every tool output passes through tool-output-sanitizer; every irreversible action requires a capability-broker token; spend-guard pauses if budget breached.

    → tool-output-sanitizer · capability-broker · guardrails-validator · spend-guard

  7. 07

    Reviewers file findings

    Code-Reviewer (correctness) and Critic (adversarial) are blocking. Verifier re-derives acceptance from a fresh worktree. Reviewer (craft) is advisory.

    → code-reviewer · critic · verifier · reviewer

  8. 08

    Steward gates the merge

    Process-graph evaluator confirms back-merge, migration breakpoints, and required contexts. Only then does the chain mark the phase done and merge.

    → steward-core · steward-analyzers

  9. 09

    Mentor remembers; Curator digests

    Any operator correction, postmerge regression, or rejected finding becomes a Mentor event. Curator rolls events into a daily digest + alarms + PR proposals + backlog directives. Surface surfaces only what the operator should actually see this week.

    → mentor-event-bus · mentor-fastpath · mentor-retrieval · curator · surface
