The platform

An operating system for autonomous engineering.

Six pillars hold up every CAIA-built feature — from intent capture to merged PR. Each one is a versioned, testable package you can swap or extend.

Autonomous chains

Multi-phase state machines with atomic lock + heartbeat + audit log. A scheduled wake dispatches the next phase; phases can be paused, resumed, adjudicated, or replayed.

@chiefaia/chain-runner

Local-first AI

A routing layer dispatches every prompt to local Ollama models first, escalating to Claude only when complexity demands it. Two-tier semantic cache cuts replay cost to zero.

@chiefaia/local-llm-router

Apprentice loop

Four-phase QLoRA pipeline: corpus → eval → train → serve. Trains a CAIA-flavoured 7B on Mac MLX from your own artifacts, with canary routing and auto-promotion gates.

@chiefaia/apprentice-*

Defense in depth

Four-layer safety stack: capability tokens for irreversible actions, input/output guardrails, tool-output sanitization, and HMAC-signed inter-service auth. Subscription-only — never sets ANTHROPIC_API_KEY.

capability-broker · guardrails-validator

Multi-agent ecosystem

PO, BA, EA, Coding, Fix-It, Steward, Mentor, Curator, Critic, Code-Reviewer, Reviewer, Verifier, Researcher, Surface, Librarian — each one a focused, fixture-tested specialist with a typed contract.

@chiefaia/claude-subagents

Event-driven memory

An append-only Mentor event bus captures every correction, postmerge regression, and operator preference. Embedded retrieval prepends precedent into every spawned worker's prompt.

mentor-event-bus · librarian

What is CAIA?

Not a chatbot. A factory.

CAIA — the Chief AI Architect — is the operator's persistent organisational substrate. It captures intent, decomposes it into auditable work, runs autonomous chains across local and cloud models, and gates every output through a four-layer review.

It's the difference between asking an LLM "write this for me" and running an engineering org. You bring the judgement. CAIA brings the execution, the memory, and the gates.

subscription-only local-first audit-by-default private workspace
  1. 01

    Operator captures intent

    Free-form prose, a backlog item, or a chat message in the dashboard.

  2. 02

    Decomposer + PO agent classify

    Initiative → Epic → Module → Story → Task. Feature registry checks if it's new or an enhancement.

  3. 03

    Phase-1 agents draft the contract

    BA, EA, DBA, BFF, UI, Security, Testing, Release each fill a Zod-validated ticket-template section.

  4. 04

    Coding agent executes in a worktree

    Spawned via the subscription-only claude-spawner. Output passes through tool-output-sanitizer before re-entering context.

  5. 05

    Four reviewers gate the PR

    Code-Reviewer (correctness), Critic (adversarial), Reviewer (craft), Verifier (fresh-worktree re-derivation). Steward enforces the merge contract.

  6. 06

    Mentor remembers

    Every correction, postmerge regression, and operator preference lands in an append-only event bus that prepends precedent into the next spawn.
