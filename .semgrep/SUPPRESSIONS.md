# Semgrep — operator-approved suppressions

Suppressions of `caia-*` rules from `.semgrep/caia-rules.yml` get
documented here. Suppressions of `auto` rules go inline as
`// nosemgrep: <rule-id> — <reason>` next to the offending line.

Format per entry:

```
## YYYY-MM-DD — <rule-id> — <short reason>
File: <path>
Why safe: <one paragraph>
Approved by: prakashgbid
PR: <link>
```
